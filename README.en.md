## About the Project

**Paw Hut** is a team-based educational frontend project. The goal was to create a modern, responsive, and user-friendly website for an animal shelter.

Users can:

- Browse pets available for adoption.
- Filter pets by categories.
- View detailed pet information in a modal window.
- Read answers to frequently asked questions.
- Explore success stories.
- Submit an adoption request through a form.

## My Team Role

In this team project, I worked as the **Team Lead** and was responsible for the **“Our Pets” / “Наші хвостики”** section.

My contribution included:

- Coordinating team workflow and organizing the development process.
- Developing the **Our Pets / Наші хвостики** section.
- Integrating external API data with Axios.
- Rendering pet cards dynamically.
- Implementing category filtering.
- Implementing pagination / “Show more” logic.
- Handling loading states and API errors.
- Connecting pet cards with the animal details modal.

## ✨ Features

- **Responsive layout:** optimized display for mobile, tablet, and desktop.
- **Dynamic pet list:** pet cards are rendered from API data.
- **Category filtering:** users can filter animals by categories.
- **Pagination / Show more:** gradual loading of additional cards.
- **Pet details:** modal window with detailed pet information.
- **FAQ section:** frequently asked questions in accordion format.
- **Success Stories:** section with successful adoption stories.
- **Adoption form:** contact form for adoption requests.
- **User notifications:** feedback and error handling.

## Tech Stack

- **Core:** HTML5, CSS3, JavaScript (ES6+)
- **Bundler:** Vite
- **API:** REST API, Axios
- **Deployment:** GitHub Pages
- **Libraries and tools:** Swiper, Accordion-js, Raty-js, SweetAlert2, SimpleLightbox, iziToast

## Installation and Running

### Prerequisites

- Installed [Node.js](https://nodejs.org/) current LTS version.

### Steps

1. **Clone the repository:**

   ```bash
   git clone <repository-url>
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the project in development mode:**

   ```bash
   npm run dev
   ```

4. **Open the link shown in the terminal**  
   usually `http://localhost:5173/`.

### Build the project

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```
