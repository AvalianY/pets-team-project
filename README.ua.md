## Про проект

**Paw Hut** — це командний навчальний frontend-проект. Мета проекту — створити сучасний, адаптивний та зручний сайт для притулку тварин.

Користувачі можуть:

- Переглядати тварин, доступних для адопції.
- Фільтрувати тварин за категоріями.
- Відкривати детальну інформацію про тварину в модальному вікні.
- Читати відповіді на часті запитання.
- Переглядати історії успішної адопції.
- Надсилати заявку через форму.

## Моя роль у команді

У цьому командному проекті я виконував роль **Team Lead** та відповідав за секцію **“Наші хвостики”**.

Мій внесок:

- Координація роботи команди та організація процесу розробки.
- Розробка секції **Наші хвостики**.
- Інтеграція зовнішнього API через Axios.
- Динамічний рендеринг карток тварин.
- Реалізація фільтрації за категоріями.
- Реалізація пагінації / логіки “Показати більше”.
- Обробка станів завантаження та API-помилок.
- Зв’язок карток тварин із модальним вікном деталей.

## ✨ Функціонал

- **Адаптивна верстка:** коректне відображення на mobile, tablet та desktop.
- **Динамічний список тварин:** картки тварин рендеряться з API-даних.
- **Фільтрація за категоріями:** користувач може фільтрувати тварин.
- **Pagination / Show more:** поступове завантаження додаткових карток.
- **Деталі тварини:** модальне вікно з детальною інформацією.
- **FAQ-секція:** часті запитання у форматі accordion.
- **Success Stories:** блок з історіями успішної адопції.
- **Форма адопції:** форма для контакту.
- **Повідомлення користувачу:** feedback та обробка помилок.

## Технологічний стек

- **Core:** HTML5, CSS3, JavaScript (ES6+)
- **Bundler:** Vite
- **API:** REST API, Axios
- **Деплой:** GitHub Pages
- **Бібліотеки та інструменти:** Swiper, Accordion-js, Raty-js, SweetAlert2, SimpleLightbox, iziToast

## Встановлення та запуск

### Передумови

- Встановлений [Node.js](https://nodejs.org/) актуальної LTS-версії.

### Кроки

1. **Клонуйте репозиторій:**

   ```bash
   git clone <repository-url>
   ```

2. **Встановіть залежності:**

   ```bash
   npm install
   ```

3. **Запустіть проект у режимі розробки:**

   ```bash
   npm run dev
   ```

4. **Відкрийте посилання, яке з’явиться у терміналі**  
   зазвичай це `http://localhost:5173/`.

### Збірка проекту

```bash
npm run build
```

### Перегляд production-збірки

```bash
npm run preview
```
